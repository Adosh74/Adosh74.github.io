 # Shortcuts
| **Shortcut** | **Description** |
| --------------|-------------------|
| `Esc` | Switch to normal mode. | 
| `i` | Switch from normal to insert mode. | 
| `k` | Move to the above line. | 
| `j` | Move to the next line. |
